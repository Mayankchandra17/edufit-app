# edufit-app
EduFit is a Flutter + Firebase based mobile application designed to help college students manage academic resources and fitness in one place.

## Features
- Department-wise subject navigation
- Notes and PYQs access
- YouTube learning resources
- Fitness plans (muscle gain, weight loss, etc.)
- User authentication using Firebase

## Tech Stack
- Flutter (Frontend)
- Firebase Authentication
- Cloud Firestore

## Project Structure
- Education Module (Departments → Semesters → Subjects)
- Fitness Module (Workout & Diet Plans)
- Profile Management
